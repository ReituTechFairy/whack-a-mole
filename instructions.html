Whack-a-Mole Arena

Game Instructions

Game Goal

The goal of the game is to click the moles as they appear before they disappear. Each successful click increases your score.

Some moles may be bonus moles and give more points than normal moles.

How to Play

  1. Enter your player name on the launcher page.
  2. Select the difficulty level.
  3. Select the game length.
  4. Choose a theme such as Classic, Farm, or Robots.
  5. Select optional features such as sound, double points, or bonus moles.
  6. Click Open Game Window.
  7. In the game window, click Start Game.
  8. Click moles as they appear in the holes.
  9. Try to score as many points as possible before time runs out.

Difficulty Levels

  • Easy: moles appear more slowly and stay visible longer
  • Medium: moderate speed and visibility time
  • Hard: moles appear faster, disappear sooner, and may appear in more than one hole at a time

Buttons in the Game Window

  • Start Game starts a new game
  • Pause / Resume pauses or resumes the game
  • Save Session stores the current game state
  • Load Session restores a saved game
  • Reset Game clears the current game
  • Back to Settings closes the game window and returns to the launcher

Scoring

  • Each normal mole adds points
  • Bonus moles add more points
  • If Double points is enabled, all successful hits score double
  • Clicks on empty holes may count as misses
  • The game ends when the timer reaches zero

Tips

  • Keep your mouse pointer near the middle of the game area
  • React quickly to bonus moles because they disappear fast
  • On hard mode, scan the whole board constantly
